# Space Invaders

A modern JavaScript recreation of the classic Space Invaders arcade game, built using HTML Canvas, JavaScript, and Vite. A rewrite of my initial version - 'invaders'.

## Live Demo

[Space Invaders](https://alphynweb.github.io/space-invaders/)

## About

This project is an implementation of Space Invaders built from scratch, designed to explore game development concepts and modern JavaScript architecture.

The game uses an object-oriented structure with separate systems responsible for game logic, graphics, input handling, collision detection, audio, and entity management etc.

## Features

- HTML Canvas rendering
- Object-oriented JavaScript architecture
- Game state management
- Collision detection system
- Sprite-based graphics
- Web Audio API sound system
- Responsive canvas scaling
- Keyboard controls

## Technologies

- JavaScript (ES Modules)
- HTML5 Canvas
- Web Audio API
- Vite
- GitHub Actions
- GitHub Pages

## Graphics & Audio
